# New tag: `<cms:md5>`

Calculates MD5 hash

## Example
```html
<cms:test
    ignore='0'
    >
  <cms:set
      myvar='example text'
    />

  <cms:md5 var=myvar />

</cms:test>
```
HTML:
```html
f81e29ae988b19699abd92c59906d0ee
```
