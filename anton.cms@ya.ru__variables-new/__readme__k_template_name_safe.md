# New variable: `k_template_name_safe`

Quickly access **safe** name of current template via &ndash;
- `k_template_name_safe` variable, or, without extension
- `k_template_name_safe_ex`

## Example
```html
<cms:test
    ignore='0'
    >
  <cms:dump_all />

</cms:test>
```
